UNIVERSAL EXTENDED (UX)
RECEIVER

Used for universal commands that make changes to the map and/or game.

!!UX:XXXX; Universal commands. These commands relate to many different things.

OPTIONS

D#;
Alpha 8.2 		Display ERM error message
    # - Error message. Might be z-var or ^^ constant
Useful for debugging. Works with UN:P904 and UN:P905.
G#/$; Get/check/set town-specific grail effect state
    # - Town (See Format T)
    $ - variable to store/set state
        $ = 1 enabled
        $ = 0 disabled
For example, UX:G8/0 will disable the Conflux grail effect of learning all spells.
K#/?$; Check if key is pressed right now
    # - key code (See Format VK)
    $ - variable to store the key state
        $ = 1 key pressed
        $ = 0 key NOT pressed
M$1/$2; Get/check/set mouse coordinates
    $1 - X coordinate
    $2 - Y coordinate
Comment:
The coordinates may be negative or more than 800,600 if the mouse is outside the Heroes window.
S#/$; Get/check/set a big brunch of global (internal) strings
    # - string index (See Format IS)
    $ - string, z-variable or z-variable index. 0 restores default.
T$; Get current time or show dialog with time delta
Returns time that ellapsed after the Windows was started in milliseconds when used with get (or check) syntax. When used with set syntax it subtracts the value passed as parameter from the current time and shows the differance in a message.

Example
!!UN:T?y10;
[some calculations that take much time...]
!!UN:Ty10;
