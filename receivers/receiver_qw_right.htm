QUEST LOG (QW)
 Receiver

Lets you add, remove or change an entry in the new Quest Log (Papyrus). 
This log is in addition to the standard Quest Log and does not replace it. 

!QW:XXXX; Quest log controller


OPTIONS

A#1/#2/#3/$4; Lets you add, change, or remove an entry in the new Papyrus quest log.
    #1 - A unique number you set for the the particular quest (set only)
    #2 - Hero number (See Format H)
        -1 - Current
        -2 - Any (all heroes). Set only
    #3 - Owner (See Format E1).
        -1 - Current, 
        -2 - Any (all colours) Set only.
    $4 - index for Z variable containing quest text. 
        -1 - Delete this quest (set/check/get).
Comments:
You can use a combination of #2=-2 and #3=-2:
 #2=-2 and #3=owner - all heroes of this owner have this quest.
 #2=hero and #3=-2 - this hero of any owner (if rehired) has this quest.
Quest text may be formatted.

To access this log, a player must click on the Quest Log button in the Hero window. First the Papyrus log will appear (unless blank) and then by clicking the OK button, the standard Quest Log will appear after. The Adventure Map Quest Log button only brings up the standard Quest Log.

 

Comments
You can use the quest log for any purpose, not just quests. It could be a diary that records interesting places the hero visits. Or you could use it to give a player hints at specific or random intervals or locations. It could even be used for humorous purposes if you wish to put little jokes in to make the player giggle. :-)
