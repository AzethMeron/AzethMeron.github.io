MP3 CONTROLLER (MP)
RECEIVER

Used to change in-game predefined music. Filepath starts in /mp3/, so you must always use paths relative to this directory.

!!MP:XXXX;

MP3 control receiver

OPTIONS

C$; Set/get/check index of incoming music.
    $ - index of upcoming music (See Format MP)
If the MP3 is NOT in the list, 0 is returned.
Use only within !?MP trigger
N#; Play MP3 file
    # - filename in z-variable or ^^ constant
    # - (alternative) index of MP3 (See Format MP)
P$; Set/get/check incoming music.
    $ - filename of incoming music in z-var or ^^ constant.
Use only within !?MP trigger
S#1/$;
 S#1/^filename^;		 Sets a custom MP3 file to play instead of predefined or user defined game music.
 # is from 1..199. (see Format MP)
  $ is the index of a z variable 

Comments:
Do not include the extension ".mp3". This is added automatically.

Example of setting a custom music file named "Sample.mp3":

ZVSE
!?OB6/6/0; // visit something at 6/6/0
// Set Castle music to Sample.mp3 in DATA folder
!!MP:S1/^..\data\Sample^;

If you put an mp3 file in the mp3 folder, you do not need to mention a folder itself:
!!MP:S1/^Sample^;

Or if you have a file in the maps folder:
!!MP:S1/^..\maps\Sample^;
