HINT DISPLAY (HD)
RECEIVER

Used for commands changing hint of any object, displayed after right-click OR mouseover in status bar.
Use only within !?HD trigger.

!!HD:XXXX; Hint Display

OPTIONS

C?$;
Check if hint of the object was customized with HT reciever or with HO:H command.
    ?$ = 0 not customized
    ?$ = 1 customized
In many cases you shouldn't process modified hints.
M$;
Get/check/set hint text
    $ - probably z-var
We must check which strings are valid here.
P#1/#2/#3/ #4/#5/#6; Set pictures for the message
    #1 - picture 1 type
    #2 - picture 1 subtype
    #3 - picture 2 type
    #4 - picture 2 subtype
    #5 - picture 3 type
    #6 - picture 3 subtype
You can use less than 6 parameters if you don't need all 3 pictures. If you use it in mouse over hint, it will be just ignored.
T?$; Check if it's a right-click hint
    ?$ = 0 means mouseover hint, to be displayed in status bar
    ?$ = 1 means right-click hint


 Example: 
!?HD;
!!OB998:T?v1 U?v2; [get type and subtype of object to which the square belongs]
!!HD&v1=5:P8/v2;
!!HD&v1=93:P8/1;
[Displays artifact picture in right-click message]
