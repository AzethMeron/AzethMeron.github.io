CASTLE INCOME (CI)
RECEIVER

Used for commands changing gold income and monster growth in Castle.
Use only within !?CI trigger.

!!CI:XXXX; Change town income/growth

OPTIONS

I$1;
Set/check/get current income (or growth)
    $1 - current income, if used within !?CI0 trigger
    $1 - current monster growth, if used within !?CI1 trigger
I$1/1;
Get/check/set base income (or growth)
    $1 - base income, if used within !?CI0 trigger
    $1 - base monster growth, if used within !?CI1 trigger
While ERM can change those values, it is recommanded to NOT do that.
L$1; Set/check/get monster level
    $1 = monster level to be used with !!CI:I receiver
    $1 = -1 if used within !?CI0 trigger.


 Example: 
!?CI;
!!CA-1:T?v1; [get town type]
!!CI&v1=1:Id500; [increase income of all Ramparts by 500]
