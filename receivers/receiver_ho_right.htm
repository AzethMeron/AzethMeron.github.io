HERO INTERACTION (HO)
RECEIVER

Used to enable/disable interaction with heroes.
When any hero attacks or visits any other !?HE trigger occurs

!!HO#:XXXX; # means a number of hero. This is not the same as the HE receiver. At present, it is solely used for enabling/disabling interaction with a given hero.

OPTIONS

D$; Disable all heroes of colour $ (-1=current) to attack (enemy hero) or visit (allied hero) this hero. See Format E1.
E$;  Enable all heroes of colour $ (See Format E1) (-1=current) to attack (enemy hero) or visit (allied hero) this hero
H$; Set a hint for this hero to z variable. (See Format E1).
R; Enable all heroes to attack (enemy hero) or visit (allied hero) this hero.
S; Disable all heroes to attack (enemy hero) or visit (allied hero) this hero
