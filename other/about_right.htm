Introduction

 ERM means Event Related Model and is dedicated to Heroes3: Shadow of Death game and add-on. ERM is a new technology that makes possible dynamically changing the map objects' properties as a result of certain player's (including computer) actions. It essentially offers scripting (programing) posibility. Some default script files ensure WOG enhancements as described in the latest version info, but ERM also gives the basic game user access to creating its own scripts to add his own imagination and flavor to the game. It would be a good idea to start with the tutorials

Does WoG work in multiplayer? 
3.58: Yes! In earlier versions, there was no network support (only single player and hotseat) but now WoG fully supports Network play including TCP/IP. The only exception is that scripts dealing directly with battle commands may not function correctly in a human vs. human battle online, however there are ERM commands now to check and adjust or disable the script accordingly.
3.59: Honestly not sure, check Engine for more informations.
