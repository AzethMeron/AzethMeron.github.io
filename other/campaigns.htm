NEW CAMPAIGNS SUPPORT

New campaigns might be added to Menu Screen using Lua scripts parsed at game start (fe. 'global' folder in mod directory). Support itself is written into 3.59 engine, script responsible for that is 'Lua/Misc/Campaign.lua'.

Ingame, it looks like this


You can programm external campaign by adding new entry to global array 'CampaignPacks'. Format: 
CampaignPacks[some_name] = { -- name=string, must be unique, so it's good idea to use ModName..smt
	Button = {X, Y, File, Name, Hint }, 	-- parameters for menu button. 
						-- X,Y are position in pixels, 
						-- File is DEF file with button
						-- Name, Hint - selfexplanatory
	Picture,  -- background picture (pcx) to be used in campaign select 
	Campaigns = { 	-- array of arrays
		{ 	-- each campaign is defined by array with given format
			File, 		-- h3c file with campaign
			Label, 		-- Name of this campaign 
			Picture, 	-- pcx file with picture
		}, 
		-- here add 6 more
	}
}
