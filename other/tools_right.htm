TOOLS

There are some useful programs that can be used to enhance gameplay experience of ease the life of mod makers, alongside with brief description and download links.


Dialog Editor
Created by GrayFace. Links: ORIGINAL MIRROR
!!DL reciever lets you create custom dialogs for WoG TE / Era / 3.59. No help for this one. Development isn't continued.

DefPreview
Created by GrayFace. Links: ORIGINAL MIRROR
Shows .def files, lets you extract frames and make a def-list for Def Tool.

DefTool
Created by GrayFace. Links: ORIGINAL MIRROR
Lets you create .def files. It can also help with editing interface pictures. Inside app you can access brief introduction.
Polish tutorial attached in MIRROR.

DefMaker
Created by ???. Links: ACIDCAVE MIRROR
Another program made to create .def files. Includes semi-automatic generator for animations.

TextEdit
Created by GrayFace. Links: ORIGINAL MIRROR
Txt tables editor for Heroes 3 and M&M. Use it to modify/create WoG Options file, creature info and many more

MMArchive
Created by GrayFace. Links: ORIGINAL MIRROR
Fully-featured M&M and Heroes 3 archives editor. Supports all archive types except ".hwl", thus includes .lod, .snd, .vid

MonPower
Created by GrayFace. Links: ORIGINAL MIRROR
Calculates strength of monsters. The output values somewhat resemble AI Value and Fight Value. Only accounts for basic parameters (ignores Speed, shooting etc.)
